문제 1 -> 1/거리^2 스케일로 축소하니 소리가 너무 작아지는 문제 발생
해결 1 -> 실제 점파원과는 다르게 단일 광선이니 1/거리 스케일로 변경
문제 1-1 -> 그럼에도 불구하고 각각의 소리의 크기가 합산시 원본 값과 괴리가 커짐
해결 1-1 -> 1밀리초 내에 들어오는 모든 소리를 통합하고 원본 소리보다 커지지 않게 조절

문제 2 -> 연속적인 실수 값을 그대로 저장하니 파일의 길이 및 연산량 폭증
해결 2 -> 시간 단위를 밀리초로 이산화하고 볼륨 스케일도 퍼밀로 표현해 이산화함.
문제 2-1 -> 밀리초로 이산화하니 위상으로 인한 간섭 표현이 어려워짐
해결 2-1 -> 2차원 벡터(거리, 위상)와 음역대별 샘플링을 통해 위상차로 인한 간섭을 구현함

문제 3 -> 시뮬레이션 완료시 나타나는 거리 변수만으로는 주파수별 감쇠 정도의 차이 구현이 어려움
해결 3 -> 주파수별로 크게 7개의 주파수로 미리 구한 뒤, 로그를 씌워 내분하여 합성 시 적용시킴
문제 3-1 -> 설정 영역 밖인 100 미만이나 6400 초과의 주파수는 내분 연산 불가
해결 3-1 -> 50, 12800의 가상 주파수를 만든 뒤 소리 값을 0으로 설정하여 연산시 사용(해당 영역 밖은 주로 기타에서 나타나지 않아 0 취급)

추가 1 -> 원하는 방 구조를 매번 제작하는게 번거로움
보완 1 -> 프리셋 저장/로드 기능 추가

문제 4 -> 리버브 적용 시 노이즈 같은 잡음에도 적용되어 소리가 지저분해짐
해결 4 -> 노이즈게이트 이펙터를 간단히 구현하여 일정 음량 이하를 미리 쳐낸 뒤 적용시킴
